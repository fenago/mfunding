## Phase 1 — Systems Access (Day 1–2)
- [ ] GHL (CRM) account created, role/permissions set, MFA enabled
- [ ] Assigned dialer + local number(s); missed-call text-back tested
- [ ] CRM training: pipelines (MCA 13-stage, VCF 8-stage), custom fields, logging notes
- [ ] Secure document-upload / Plaid link walkthrough
- [ ] Access to approved scripts, disclosures, and offer-comparison templates only

## Phase 2 — Product & Compliance Training (Day 2–4)
- [ ] MCA fundamentals: factor rate, total payback, retrieval rate, points/commission
- [ ] **Language rules**: MCA is never a "loan"; correct terms for each product
- [ ] No upfront fees; "no credit impact" to review (soft) vs. formal submission
- [ ] State disclosure process: which states, which doc, when to deliver
- [ ] TCPA/DNC: consent basis, quiet hours, opt-out handling, scrubbed-lists-only
- [ ] VCF program: attorney-led restructuring + FDIC refinance; no guarantees; not a law firm; substantiation of "50–75%" range
- [ ] Funder network overview + submission/stip requirements
- [ ] Clawback mechanics and why deal quality matters

## Phase 3 — Sales Skills & Ramp (Week 1–2)
- [ ] Shadow live calls / call recordings
- [ ] Role-play: speed-to-lead (<60 sec), qualification (BANT-F), objection handling
- [ ] Stip/bank-statement chase (the #1 funnel leak) and follow-up sequences
- [ ] First supervised live calls; manager QA on first deals
- [ ] Draw terms confirmed (if elected)

## Phase 4 — Go Live
- [ ] Added to lead round-robin
- [ ] First-week targets set (calls, apps sent, docs collected)
- [ ] Weekly 1:1 cadence scheduled
- [ ] 30/60/90-day review dates set

## Ongoing Compliance
- [ ] Periodic call QA / script-adherence review
- [ ] Re-sign acknowledgments on material policy changes
- [ ] Opt-out and complaint handling refresher quarterly
